# 标签速查（官方包 `official-001`）

包路径：`$MEME_PACK` 或 `${MEME_HOME}/meme-packs/official-001`（兼容 `HERMES_*`）

来源建议：[astrbot-meme-pack-official-01](https://github.com/anka-afk/astrbot-meme-pack-official-01)

| 标签 | 场景 |
|------|------|
| `happy` | 庆祝、好消息、任务完成 |
| `like` | 喜欢、夸赞、觉得不错 |
| `meow` | 卖萌、轻松安抚 |
| `fool` | 自嘲、小失误、缓和气氛 |
| `baka` | 可爱型吐槽、轻微责备 |
| `confused` | 没听懂、逻辑对不上 |
| `cpu` | 技术问题卡住、需要想想 |
| `sad` | 安慰、遗憾、坏消息 |
| `sigh` | 无奈、无语 |
| `sleep` | 熬夜、累了 |
| `surprised` | 意外、没想到 |
| `shy` | 被夸、隐私话题 |
| `see` | 偷瞄、盯进度 |
| `morning` | 早安 |
| `reply` | 提问后等用户回 |
| `work` | 下班/上班、工作吐槽 |
| `angry` | 傲娇、互怼、对事发火（对方真生气时慎用） |
| `color` | 暧昧调情（默认慎用） |
| `givemoney` | 谈钱/讨赏 |

完整描述以包内 `manifest.json` 为准：

```bash
python3 scripts/pick-meme.py --list
```
